## 1. 我们收集的信息
1.1 设备权限相关信息

为了提供本 App 的核心功能，我们会在获得用户授权后访问以下设备资源：
	•	**摄像头（Camera）：**用于实时 AR 视频捕捉及生成动作数据。
	•	**麦克风（Microphone）：**用于录制视频音频、声音触发动画、语音控制模型功能。
	•	**相册（Photo Library）：**用于将用户录制的视频保存至用户设备相册。
	•	**语音识别（Speech Recognition）：**用于在 AR 场景中识别用户的语音指令并触发相应动作。

我们不会在未征得用户许可的情况下访问上述权限。

## 2. 我们不收集、不存储的用户信息
除非用户主动上传资源至数据库，否则：
	•	我们不收集用户的照片、视频、音频或任何相册内容。
	•	我们不存储用户的录制视频，这些视频仅保存在用户设备上。
	•	我们不进行人脸识别、行为分析或个人身份识别处理。
	•	我们不向第三方出售或共享用户的个人数据。
